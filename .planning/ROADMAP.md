# Roadmap

## Active Milestone

- `M1-smart-correction-history`: Smart correction learning from transcript history.

## Future Backlog

Future work can extend this foundation into broader Wispr Flow-style capabilities:

- snippets triggered by spoken phrases;
- app-aware post-processing styles;
- command mode for rewrite/summarize/edit instructions;
- multi-device sync for dictionary and snippets;
- team/shared dictionary.

These are out of scope for M1.
